# git-sync

[![CI Build](https://github.com/joshburnsxyz/git-sync/actions/workflows/ci_build.yml/badge.svg)](https://github.com/joshburnsxyz/git-sync/actions/workflows/ci_build.yml)

Keep your projects folder up-to-date.

## Usage

__TL;DR__

```console
$ cd ./my_git_project
$ git status
$ git sync
```

_or_

```console
$ ~/my_projects_folder
$ git sync -r
```
